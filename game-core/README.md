# GAME CORE — PROJECT RAVY

Motor principal del videojuego PROJECT RAVY.

## Contenido
- Sistema de disparos (FPS)
- Battle Royale
- Multijugador online
- IA avanzada (RAVY)
- Sistema de niveles
- Armas realistas → futuristas
- Mapas híbridos (ciudad real + mundo futurista)

## Tecnologías previstas
- Motor gráfico: Unity / Unreal
- Lenguaje: C# / C++
- Red: Servidores dedicados
- Física realista

## Objetivo
Crear un shooter competitivo de nueva generación para móvil y PC.
